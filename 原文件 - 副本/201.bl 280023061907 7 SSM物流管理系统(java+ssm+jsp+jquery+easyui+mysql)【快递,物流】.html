SSM的物流管理系统

系统主要分为三种用户角色，分别是管理员、员工以及客户，其具体功能如下：

管理员：个人信息、员工管理、客户管理、反馈信息、基础信息、货物信息、货物运输、统计信息、系统管理
员工：个人信息、客户管理、反馈信息、基础信息、货物信息、货物运输
客户：个人信息、配送范围、货物信息、货物运输、反馈信息

后台框架：Spring、SpringMVC、MyBatis
UI界面：JSP、jQuery、EasyUI
数据库：MySQL
